# Portfólio Dev — Martina Santos

Página de portfólio pessoal desenvolvida com HTML e CSS puro, apresentando projetos, serviços e informações de contato de uma desenvolvedora Fullstack.

## Seções

- **Apresentação** — foto, nome, descrição e stack de tecnologias
- **Meu trabalho** — grade com 6 projetos em destaque
- **Meus serviços** — cards de Websites & Aplicativos, API & Banco de Dados e DevOps
- **Contato** — links para redes sociais com hover animado

## Tecnologias

- HTML5
- CSS3 (variáveis, Flexbox, Grid, transições)
- Fontes: Asap, Inconsolata, Maven Pro (Google Fonts)

## Estrutura

```
index.html
assets/
  icons/
  images/
styles/
  global.css    # reset, variáveis CSS e estilos base
  index.css     # imports
  home.css      # estilos das seções da página
  utility.css   # classes utilitárias de tipografia e cor
```

## Como rodar

Abra o arquivo `index.html` diretamente no navegador ou use a extensão **Live Server** no VS Code.
